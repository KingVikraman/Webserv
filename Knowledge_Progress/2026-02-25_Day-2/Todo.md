**Day 2-3: Building Your First Socket Server**

This is where the real fun begins - you'll write actual C++ code that:
1. Creates a socket
2. Listens for connections
3. Accepts a client
4. Echoes back whatever they send

**It's about 50 lines of code and will take ~1 hour.**

You need to understand how sockets work BEFORE building the real thing.

Think of it like:
- Learning to ride a bicycle with training wheels
- THEN riding a real bicycle
